# AquaBlast Pressure Washing Template

Next.js marketing site template for a Central Texas pressure washing / exterior cleaning business.

**Business:** AquaBlast Pressure Washing  
**Domain:** aquablastpw.com  
**Phone:** (254) 900-8842  
**Accent:** Teal `#0891b2`

## Features
- Public marketing site (services, industries, blog, projects, service areas)
- Lead-capture contact forms with reCAPTCHA
- Admin dashboard (leads, sessions, analytics)
- Per-city SEO subpages under each service

## Dev
```bash
npm install
npm run dev
npm run typecheck
npm run lint
```

See `docs/superpowers/` for design spec and implementation plan.
